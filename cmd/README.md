# Command examples

## Playback

```bash
./controller playback (song|setlist) song-or-setlist-uuid \
    --data-dir /var/controller/defintions/ \
    --wait-for-trigger audio \
    --trigger-audio-freq 15000 \
    --transport midi \
    --midi-device-id 1 \
    --transport artnet
```
